# TreeGraft

TreeGraft is a free, local-first Git GUI with structural merge intelligence — it understands code at the level of functions, classes, and entities rather than raw lines, automatically resolves independent changes, and explains every merge decision visually, all without files ever leaving your machine.

For full product context, architecture decisions, and coding standards see [CONTEXT.md](./CONTEXT.md). For the feature roadmap and build order see [ROADMAP.md](./ROADMAP.md).
